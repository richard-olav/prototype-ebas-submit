Assisting scientists with discovering and accessing atmospheric data

The ACTRIS DC is designed to assist scientists with discovering and accessing atmospheric data and contains an up-to-date catalogue of available datasets in a number of databases distributed throughout the world.

Learn more

We continue to expand

A site like this can never be complete, but we have aimed at including datasets from the most relevant databases to the ACTRIS project, also building on the work and experiences achieved in the EU FP6 research project Global Earth Observation and Monitoring.

Providing data

The focus of the web portal is validated data, but it is also possible to browse the ACTRIS data server for preliminary data (rapid delivery data) through this site.

Discover and download across data archives

The "Data discovery and download" mode allows for browsing all available ACTRIS data, and also relevant data from other networks and frameworks as GAW-WDCA, GAW-WDCGG, EMEP, INGOS, EARLINET and downloading the requested data sets via the data portal.

Start discovering

Online analysis and plotting of ACTRIS data

The "Online analysis and plotting" mode filters the available data to ACTRIS data and other data stored in the ACTRIS data repositories, EBAS, EARLINET DB and Cloudnet DB. These can be accessed directly by the data portal. Using the "Online analysis and plotting" mode you can analyse all ACTRIS measurement data online, create graphical output via the data portal, and download.

Start analyzing the data

Near-Real-Time showcase

The Near-Real-Time showcase application gives you access to the latest NRT data readily available as plots with links to the actual data. For now we present NRT data related to EBAS and cloudnet.
